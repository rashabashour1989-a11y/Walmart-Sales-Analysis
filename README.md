# Walmart Sales Analysis Dashboard 📊

## Project Overview
This project presents a comprehensive sales analysis for **Walmart**, built using **Power BI Desktop**. The dashboard provides actionable insights into weekly sales trends, store performance, and the impact of holidays on overall revenue.

## Key Features
* **Interactive Time Intelligence:** Filter sales data by year (2010, 2011, 2012) to observe growth and seasonal trends.
* **Store Performance Analysis:** A detailed bar chart comparing sales across different store locations.
* **Holiday Impact Study:** A specialized visualization (Pie Chart) comparing sales performance during holiday weeks vs. normal weeks.
* **KPI Tracking:** Real-time calculation of Total Sales ($6.74bn overall).
* **Professional UI/UX:** Applied custom themes, rounded corners, and shadows for a modern, clean look.

## Tools & Skills Used
* **Power BI Desktop:** Data visualization and dashboard design.
* **Data Transformation:** Cleaning and formatting the Walmart dataset.
* **DAX (Data Analysis Expressions):** Used for creating measures and calculating sums.
* **Dashboard Design:** Implementing professional formatting (Visual Borders, Data Labels, and Effects).
